# licenciementeco

Guide pédagogique pour aider les salariés à comprendre le **licenciement économique**
et surtout le **Contrat de sécurisation professionnelle (CSP)** en droit du travail français.

## Contenu

- [`index.html`](./index.html) — page web autonome (guide visuel destiné aux salariés).
  Ouvrez-la dans un navigateur, ou publiez-la telle quelle.

## Points couverts

- Ce qu'est un licenciement économique (art. L1233-3 du Code du travail) et vos protections
  (reclassement, ordre des licenciements, priorité de réembauche).
- Le déroulé du CSP étape par étape (information, délai de réflexion de 21 jours, décision,
  accompagnement, 12 mois de sécurisation).
- Le comparatif **accepter vs refuser** le CSP.
- L'allocation **ASP** (75 % du salaire brut de référence) et le traitement du préavis.
- Les aides au reclassement : prime au reclassement, indemnité différentielle (IDR),
  périodes d'activité, formation.
- Une FAQ et les sources officielles.
